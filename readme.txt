Another last posts widget which get recent posts only for the current root category.

== Description ==

In order to work, you will need the library "Category class" : http://wordpress.org/plugins/category-class/

This widget will find the root category which the current post (or category) depends.
Once done, it will display the latest posts from the root category, not for the entire site.

Based on the "Recent post" widget, you will find the same parameters with a new checkbox : you will be able to add the category name in titles.

You can find a demo here : http://www.spoils.fr/
The widget is named "Articles Recents" on the sidebar.

== Installation ==

= Manual installation =

1. Upload folder into to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Use your 'Presentation'/'Sidebar Widgets' settings to drag and configure

= Automatic installation =

1. Login to the admin interface.
1. Click Plugins in the left hand menu navigation.
1. Click on "Add New" next to the Plugins header in the main content area.
1. Enter "Root Category Recent Posts" (without quotes) in the textbox and click the "Search Plugins" button.
1. In the list of relevant plugins click the "Install" link for "Root Category Recent Posts" on the right hand side of the page.
1. Click the "Install Now" button on the popup page.
1. Click "Activate Plugin" to finish installation.
1. Use your 'Presentation'/'Sidebar Widgets' settings to drag and configure
